👥 Proto-personas / Actores del Sistema y Código de Color

Ana, Administrativa Contable

Flujo Operativo
  • Objetivo principal: Digitalizar facturas rápidamente sin carga manual redundante.
  • Frustración clave: Pérdida de tiempo tipeando datos campo por campo y corrigiendo errores de transcripción.
  • Nivel técnico: Intermedio (manejo fluido de planillas de cálculo y sistemas web de gestión).
  • Foco en el mapa: Carga de documentos, control del OCR, edición y persistencia de registros diarios.

Carlos, Analista de Datos / Auditor

Flujo Analítico
  • Objetivo principal: Consultar el historial y descargar reportes consolidados para auditorías y análisis de costos.
  • Frustración clave: Dificultad para buscar información histórica de proveedores o cruzar datos en formatos heterogéneos.
  • Nivel técnico: Avanzado (procesamiento de datos y exportación de estructuras limpias hacia herramientas BI).
  • Foco en el mapa: Explotación de la información histórica, filtros combinados complejos y exportación limpia para auditorías.
